# 🤖 AI Skills

Коллекция скиллов для [Claude AI](https://claude.ai) (Cowork mode) — инструменты для автоматизации повседневных задач.

## Что такое скиллы?

Скиллы — это расширения для Claude, которые дают ему специализированные возможности. Каждый скилл содержит `SKILL.md` с инструкциями и, при необходимости, вспомогательные скрипты.

## Скиллы

### 🧹 [cleanup-folder](./cleanup-folder)
Анализ и очистка любой папки: поиск дублирующихся файлов (по MD5-хешу) и установщиков (DMG/PKG). Безопасно перемещает файлы в Корзину через macOS Finder.

**Триггеры:** «очисти папку», «найди дубликаты», «освободи место», «удали установщики», «clean up folder», «free up space»

## Использование

1. Установи [Claude Desktop](https://claude.ai/download)
2. Включи Cowork mode
3. Скопируй папку нужного скилла в `~/.claude/skills/`
4. Перезапусти Claude
